About

Virtual Class Venture is an online learning platform. Our platform is combined solution of video streaming and classroom management. We are recommending it for educational institution. It will help to conduct educational activities of an educational institution in online as well as offline.

How Teacher will use

  • Complete registration by using Email & Password.
  • Setup Profile.
  • After profile setup then login using email & password.
  • Create Classroom by giving course code, course title and a short description.
  • Can see created class in Teaching Classes section.
  • Inside on created classes teacher can take live class.
  • Teacher can save the recorded classes on "Archive Video".
  • Teacher can create assignment for student.
  • "Resources", where teacher provide all type of class related resources.
  • Teacher can make exam for student.
  • Teacher can see student list.
  • Teacher can create annoucement for student.

How Students will use

  • Complete registration by using Registraion Number & Password.
  • Setup Profile.
  • After profile setup then login using Registraion Number & password.
  • Students will be able to search his/her desired class using filtering, then they will enroll the class.
  • Can see Enrolled classes in Enrolled Classes Menu Option.
  • Inside on any Enrolled classes Student can join live class.
  • Student can access assignment given by teacher. And they aslo submit there assignment on it
  • In the "Student Resources" option, student will access all type of class resources and can comment on it.
  • Student Will be attend quiz exam on "Exam" option
  • Student can see other participate student on the class.
  • Student can see all announcement given by teacher and can comment on it.
